% ══ COL 1: BACKGROUND + CONTRIBUTIONS + REFERENCES ══

\posterbox[
  redcard, title={\bfseries Background}
]{name=background, column=1, between=row6 and row11}{
  \fontsize{30}{40}\selectfont
  Empirical risk minimization on \emph{spuriously-correlated} data
  latches onto majority shortcuts.\\[4pt]
  Mainstream remedy: \textbf{bi-level meta-learning}
  (Ren 2018; FORML) -- implicit-differentiation heavy.\\[4pt]
  \colorbox{secondary!20}{\parbox{0.94\linewidth}{\fontsize{28}{36}\selectfont
  \textbf{Wang et al. 2024} derived a \emph{closed-form} 2nd-order
  In-Run Shapley estimator -- but used only \textbf{post-hoc for
  filtering}.\par}}
  \vspace{4pt}
  \textbf{Question:} can the same closed-form formulas serve as an
  \emph{in-training} reweighter? Does the 2nd-order cross-term
  contribute a \emph{stand-alone} benefit?
}

\posterbox[
  darkcard, title={\bfseries Contributions}
]{name=contributions, column=1, between=row11 and row16}{
  \fontsize{30}{40}\selectfont
  \begin{enumerate}[itemsep=14pt]
    \item Adapt closed-form In-Run Shapley as a 5-step in-training reweighter.
    \item \textbf{RMS-rescale the cross-term} -- prevents divergence under strong spurious bias.
    \item Statistically isolate a \textbf{2nd-order benefit} on Colored MNIST ($+4.7$\,pp, $p\!=\!0.033$).
    \item Honestly characterize the \textbf{regime boundary} (works on from-scratch / small-model; fails on pretrained ResNet).
  \end{enumerate}
}

% ══ COL 2: METHOD ══

\posterbox[
  darkcard, title={\bfseries Method: Fairds}
]{name=method, column=2, between=row6 and row14}{
  \fontsize{28}{36}\selectfont
  Per-step on minibatch $\mathcal{B}_t$ with anchor $\mathcal{D}_{\mathrm{val}}$:\\[4pt]
  \colorbox{accent!20}{\parbox{0.94\linewidth}{\fontsize{26}{32}\selectfont\centering
    $\displaystyle\phi_i^{(2)} = \langle g_i,\, g_{\mathrm{val}}\rangle
    \;-\; \alpha\cdot\langle g_i,\, H_{\mathrm{val}}\, g_{\mathrm{val}}\rangle$}}
  \vspace{6pt}
  \begin{itemize}[itemsep=8pt]
    \item $g_i$ via \texttt{torch.func.vmap(grad)} (single graph).
    \item $H_{\mathrm{val}} g_{\mathrm{val}}$ via Pearlmutter HVP.
    \item EMA-accumulate $\phi_i$, softmax-weight, weighted SGD.
  \end{itemize}\vspace{4pt}
  \vspace{4pt}
  \colorbox{secondary!20}{\parbox{0.94\linewidth}{\fontsize{26}{32}\selectfont
    \textbf{Algorithmic contribution:} cross-term must be RMS-rescaled to first-order's magnitude. Without it Fairds-2 collapses to chance under strong spurious bias.}}
}

\posterbox[
  redcard, title={\bfseries Mechanism (E1b probe)}
]{name=mechanism, column=2, between=row14 and bottom}{
  \fontsize{28}{36}\selectfont
  Controlled spurious-feature MLP, 2 of 8 dims informative,
  spurious strength $s\!=\!10$.\\[4pt]
  \begin{itemize}[itemsep=6pt]
    \item Mann--Whitney $U(\mathrm{maj}\!<\!\mathrm{min}\,\phi)$:\\
          $p\!=\!1.89\!\times\!10^{-8}$ at imbalance $0.9$.
    \item $\Delta\phi$ monotone $\downarrow$ with imbalance.
  \end{itemize}\vspace{4pt}
  \colorbox{secondary!18}{\parbox{0.94\linewidth}{\fontsize{24}{30}\selectfont
    Cross-term penalises majority gradients aligned with $H_{\mathrm{val}}$'s dominant directions.}}
}

    No-EMA variant doesn't cure it -- underlying SNR is poor.
  }}\\[2pt]
  \colorbox{primary!18}{\parbox{0.94\linewidth}{\fontsize{24}{30}\selectfont
    Practical recommendation: \textbf{bi-level meta-learning} for
    pretrained fine-tuning regimes.
  }}
}

\posterbox[
  highlightcard, title={\bfseries Takeaways}
]{name=takeaways, column=4, between=row14 and bottom}{
  \fontsize{30}{40}\selectfont\color{textdark}
  \begin{enumerate}[itemsep=12pt]
    \item Closed-form 2nd-order Shapley reweighting is a
          \textbf{regime-dependent} tool.
    \item It works on \textbf{from-scratch / spurious-correlation}
          regimes -- with statistical isolation of the 2nd-order benefit.
    \item It \textbf{fails on pretrained fine-tuning} -- bi-level
          methods stay stronger there.
    \item RMS-rescaling of the cross-term is required for stability.
  \end{enumerate}
}
