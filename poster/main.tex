% PROBLEM
\posterbox[
  redcard, title={\bfseries Problem We Tackle}
]{name=problem, column=1, between=row4 and row11}{
  \fontsize{28}{38}\selectfont
  ERM on \textbf{spuriously-correlated} data latches onto the majority
  \emph{shortcut signal} (e.g.\ the color channel in Colored MNIST),
  collapsing minority / OOD worst-group accuracy.\\[6pt]

  \textbf{Limitations of standard remedies.}
  \begin{itemize}[itemsep=4pt]
    \item Bi-level meta-learning (FORML): per-minibatch inner
          SGD + implicit differentiation \textemdash{}
          implementation-heavy and costly.
    \item GroupDRO / IRM: requires \textbf{group labels} during training.
    \item JTT: 2-stage ERM, no intervention into the training
          \emph{dynamics}.
  \end{itemize}
  \vspace{2pt}
  \colorbox{secondary!20}{\parbox{0.94\linewidth}{\fontsize{26}{34}\selectfont
    \textbf{Wang et al.\ 2024} derived a \emph{closed-form} 1st-/2nd-order
    In-Run Shapley estimator from a single training run \textemdash{}
    but used it only for \textbf{post-hoc filtering}.}}
  \vspace{6pt}

  \textbf{Our question:} can the same closed-form formula be
  \emph{repurposed as an in-training per-sample reweighter}, and does
  the 2nd-order cross-term contribute a \textbf{stand-alone benefit}
  over the 1st-order alone?
}

% METHOD
\posterbox[
  darkcard, title={\bfseries Method: Fairds}
]{name=method, column=1, between=row11 and row19}{
  \fontsize{26}{34}\selectfont
  Parameters $\theta_t$, minibatch $\mathcal{B}_t$, anchor
  $\mathcal{D}_{\mathrm{val}}$ (50:50 group-balanced).
  At each step:\vspace{4pt}

  \textbf{\stepbadge{1} Per-sample $g_i$} via \texttt{vmap(grad)}.\\[4pt]
  \textbf{\stepbadge{2} Validation $g_{\mathrm{val}}$ + Pearlmutter HVP} for $H_{\mathrm{val}}\,g_{\mathrm{val}}$.\\[4pt]
  \textbf{\stepbadge{3} Closed-form Shapley score:}\\[2pt]
  \colorbox{accent!20}{\parbox{0.94\linewidth}{\fontsize{24}{32}\selectfont\centering\vspace{3pt}
    $\displaystyle \phi_i^{(1)} = \langle g_i,\, g_{\mathrm{val}}\rangle,\quad
     \phi_i^{(2)} = \phi_i^{(1)} - \alpha\,\widetilde{c}_i$\\[2pt]
    $c_i = \langle g_i,\, H_{\mathrm{val}}\, g_{\mathrm{val}}\rangle,\quad
     \widetilde{c}_i = c_i \cdot \tfrac{\mathrm{RMS}(\phi^{(1)})}{\mathrm{RMS}(c)+\epsilon}$\vspace{3pt}}}
  \\[4pt]
  \textbf{\stepbadge{4} EMA accumulate} $\overline{\phi}_i$ ($\beta\!=\!0.9$).\\[4pt]
  \textbf{\stepbadge{5} Softmax-weighted SGD} with quantile clip.\\[6pt]
  \colorbox{secondary!20}{\parbox{0.94\linewidth}{\fontsize{24}{32}\selectfont
    \textbf{Algorithmic contribution:} RMS rescaling of the cross-term \textemdash{} without it the 2nd-order variant collapses under strong spurious bias.}}
}

% MECHANISM
\posterbox[
  bluecard, title={\bfseries Mechanism}
]{name=mechanism, column=1, between=row19 and row24}{
  \fontsize{26}{34}\selectfont
  Controlled spurious-feature MLP, 2 of 8 input dims informative.\\[4pt]
  \begin{itemize}[itemsep=4pt]
    \item Mann--Whitney $U(\mathrm{maj}\!<\!\mathrm{min}\,\phi)$: $p\!=\!1.89\!\times\!10^{-8}$ at imbalance $0.9$.
    \item $\Delta\phi$ monotone decreasing as imbalance grows.
  \end{itemize}\vspace{2pt}
  \colorbox{secondary!18}{\parbox{0.94\linewidth}{\fontsize{22}{30}\selectfont
    Cross-term penalises majority gradients aligned with $H_{\mathrm{val}}$'s dominant directions \textemdash{} direct mechanism evidence.}}
}

% APPROACH (right-top, alongside title)
\posterbox[
  redcard, title={\bfseries How We Approach It}
]{name=approach, column=2, between=top and row11}{
  \fontsize{28}{38}\selectfont
  \textbf{Idea 1 — Post-hoc filtering $\to$ in-training weights.}\\
  Convert per-sample Shapley estimates into softmax weights every
  minibatch and run weighted SGD, instead of using them after training.
  \\[8pt]

  \textbf{Idea 2 — Geometric meaning of the cross-term.}\\
  1st-order $\langle g_i, g_{\mathrm{val}}\rangle$: gradient alignment.\\
  2nd-order $\langle g_i, H_{\mathrm{val}}\, g_{\mathrm{val}}\rangle$:
  extra penalty on majority gradients aligned with \emph{high-curvature
  directions of the validation loss} \textemdash{} curvature-aware
  down-weighting.\\[6pt]

  \colorbox{accent!18}{\parbox{0.94\linewidth}{\fontsize{26}{34}\selectfont
    \textbf{Idea 3 — RMS rescaling.} The Hessian spectrum shifts by
    orders of magnitude during training; rescale the cross-term to the
    1st-order's RMS at every step to prevent divergence.}}
  \vspace{8pt}

  \textbf{Idea 4 — EMA accumulation.}\\
  Per-step $\phi_i$ is noisy; smooth across iterations before applying
  softmax weighting.\vspace{4pt}

  \begin{itemize}[itemsep=4pt]
    \item No implicit differentiation \textemdash{} single forward
          + backward + HVP.
    \item Group labels are used \emph{only} to balance the anchor
          (never inside training).
  \end{itemize}
}

\posterbox[

  \fontsize{26}{34}\selectfont

  \posterfig{0.70}{figures/fig_waterbirds.pdf}{All Fairds variants $\le$ Vanilla; FORML dominates.}
  \vspace{1pt}

    \textbf{Recommendation:} use \textbf{bi-level meta-learning} for pretrained fine-tuning regimes.}}
}

% TAKEAWAYS
\posterbox[
  highlightcard, title={\bfseries Takeaways}
]{name=takeaways, column=2, between=row32 and row36}{
  \fontsize{28}{38}\selectfont\color{textdark}
  \begin{enumerate}[itemsep=6pt]
    \item Closed-form 2nd-order Shapley reweighting is a \textbf{regime-dependent} tool.
    \item Works on \textbf{from-scratch / spurious-correlation} regimes; the cross-term's contribution is statistically isolated ($+4.7$\,pp, $p\!=\!0.033$).
    \item \textbf{Fails on pretrained fine-tuning} \textemdash{} bi-level meta-learning remains stronger.
    \item \textbf{RMS rescaling} of the cross-term is the key to stability.
  \end{enumerate}
}
